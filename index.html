What's this?

Advent of Code is a yearly advent calendar offering programming puzzles to solve each December. Anyone can participate, and it'd great to try out new things or simply have fun.

This is a page regrouping all my solutions to the Advent of Code challenges. The solutions were developped with p5.js, a simple javascript graphical lobrary. I challenged myself to graphically represent the puzzle being solved each time. More information on my approach can be found here.

The solutions below are sorted by year. For each you can see a few tags, whether my solution is complete, if I think it's visually interesting and sometimes additional notes. There is also a link to the source code on each page, or you can access it all here.

You are free to use my pages templates if you want to create your own solutions as long as you link back to the Github page. Enjoy! :)
